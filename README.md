# Список необходимых консольных команд для новичка
---
### Работа с файлами  


* **pwd** - узнать всое расположение
* **cd %имя папки%** - перейти в следующую папку *(можно указать путь)*
* **ls** - вывести содержимое директории
* **touch** - создать файл
* **mkdir** - сщздать директорию
* **cp** - копировать
* **mv** - перенос
* **cat** - чтение текстовых файлов
* **rm** - удалить файл
* **rmdir** - удалить папку
---
### Работа с репозиторием  

* **git init** - сделать папку репозиторием
* **git status** - проверить состояние репозитория
* **git add** - вкл отслживание файла
* **git commit** - выполнить коммит
* **git log** - история коммитов
* **git remove add** - привязать удаленный репозиторий к локальному
* **git remove -v** - убедиться что репозитории связаны
* **git push** - запушить изменения на удаленный репозиторий
---  
# Статусы файлов в Git  

**Untracked** - новый файл, который еще не был добавлен через git add  
**Staged** *(indexed, cashed)* - состояние после git add, но до коммита  
**Tracked** - все файлы, в которых Git так или иначе отслеживает изменения  
**Modified** - измененный отслеживаемый файл
